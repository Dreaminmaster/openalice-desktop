## Our Compliance

As a derivative work distributing OpenAlice, we comply with AGPL-3.0 requirements:

### 1. Source Code Availability

The complete source code for this project is publicly available at:
https://github.com/Dreaminmaster/openalice-desktop

### 2. License Preservation

- The upstream AGPL-3.0 LICENSE is preserved in the OpenAlice vendor directory
- This repository includes its own LICENSE file (AGPL-3.0)
- NOTICE.md documents all third-party components

### 3. Modification Tracking

Changes to upstream OpenAlice code (if any) are tracked in:
- `patches/` directory for patch files
- `OPENALICE_UPSTREAM.md` for commit hashes and change logs

### 4. Network Use Disclosure

Under AGPL-3.0 Section 13, if you modify OpenAlice and run it as a network service, you must make the complete corresponding source code available. Since this is a desktop application running locally, this primarily applies to any modifications made to the OpenAlice source itself.

### 5. Attribution

The following attributions are included:
- `README.md`: Product description and upstream link
- `OPENALICE_UPSTREAM.md`: Upstream project details
- `NOTICE.md`: Component licensing table
- `THIRD_PARTY_NOTICES.md`: Third-party license summaries
- `About` page in the app: Version info and upstream attribution

## Desktop Wrapper License

We recommend the desktop wrapper also use AGPL-3.0 for simplicity and compliance. This creates a clear chain of license inheritance.

## If You Distribute This App

When distributing compiled binaries (DMG):

1. Include this LICENSE file
2. Include NOTICE.md and THIRD_PARTY_NOTICES.md
3. Provide a link to the source code repository
4. Document any patches applied to upstream OpenAlice
5. Preserve all copyright notices
